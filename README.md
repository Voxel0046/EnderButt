# EnderButt

Ride your ender pearl in style on Paper 1.21+ hub servers.

## Features

- Ride thrown ender pearls
- Give custom EnderButt item on join
- Configurable hotbar slot on join
- Custom item name and lore
- Hex color support (`#RRGGBB` and `&#RRGGBB`)
- Configurable launch particle + sound
- Cooldown in **seconds**
- Optional no pearl damage / no teleport damage
- Lock item in inventory (prevent drop/move)

## Requirements

- **Minecraft/Paper:** 1.21+
- **Java:** 17+
- **Build tool:** Maven 3.8+
